# molochnyi-shliakh
🧀 Molochnyi Shliakh
Лендінг-сторінка для артизанської сироварні «Молочний Шлях».
Проєкт створений як навчальна робота для демонстрації розробки сучасного користувацького інтерфейсу веб-сайту.
Опис проєкту
Molochnyi Shliakh — це односторінковий сайт (landing page), який представляє сироварню та її продукцію.
Сайт створений для демонстрації:
структури лендінг-сторінки
сучасного дизайну
базової взаємодії користувача з сайтом
адаптивної верстки
На сторінці представлено:
головний банер
інформацію про сироварню
асортимент сирів
переваги продукції
форму для замовлення
контактну інформацію
⚙️ Використані технології
Проєкт реалізований за допомогою базових веб-технологій:
HTML5 — структура сторінки
CSS3 — стилізація та адаптивний дизайн
JavaScript — інтерактивні елементи сторінки
Google Fonts — підключення веб-шрифтів
